# Soporte

## Camino rápido

- **Bug** → abrí un issue con el template de bug
- **Nueva idea o mejora** → abrí un issue con el template de feature
- **Problema de seguridad** → seguí `SECURITY.md` y no lo publiques
- **Dudas para contribuir** → leé `CONTRIBUTING.md`

## Antes de pedir ayuda

Intentá incluir:

- qué querías hacer
- qué esperabas que pasara
- qué pasó realmente
- pasos para reproducir
- logs, errores o capturas si suman contexto

## Dónde va cada cosa

| Necesidad | Canal |
|-----------|-------|
| Reportar un bug | GitHub Issues |
| Proponer una mejora | GitHub Issues |
| Consultar cómo contribuir | `CONTRIBUTING.md` |
| Reportar una vulnerabilidad | `SECURITY.md` |

## Tiempos de respuesta

Este repo se mantiene con mejor esfuerzo, así que los tiempos pueden variar según disponibilidad del equipo.

Si querés ayudarnos a responder más rápido, mandá pedidos concretos y con suficiente contexto.
